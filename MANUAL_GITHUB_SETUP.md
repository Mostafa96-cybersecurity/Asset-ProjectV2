#### Install Git First:
1. Download from: https://git-scm.com/download/win
2. Install with default settings
3. Restart PowerShell

#### Manual Commands:
```powershell
# Configure Git (first time only)
git config --global user.name "Mostafa96-cybersecurity"
git config --global user.email "your-email@example.com"

# Initialize repository (if needed)
git init

# Add all files
git add .

# Create commit with your enhanced features
git commit -m "🚀 Enhanced Asset Management System v2.0

✅ Features: 100% Hardware Collection, Graphics Cards, Monitors, Disk Formatting
⚡ Performance: Database optimized from 198.3s to <30s
📊 Database: 467 columns, duplicate-free, enhanced schema
🎯 Ready for production deployment"

# Connect to your GitHub repository
git remote add origin https://github.com/Mostafa96-cybersecurity/Asset-Project-Enhanced.git

# Push to GitHub
git branch -M main
git push -u origin main
```
